Γεννήτρια Άρθρων

0 δείκτες • 0 εικόνες
Μορφή: απλό κείμενο με επικεφαλίδες σε στυλ markdown. Οι σύνδεσμοι σε YouTube, X, Instagram, Threads και Facebook ενσωματώνονται αυτόματα.
Κεντρική / thumbnail (εικόνα 1): εμφανίζεται στις κάρτες του blog και, αν δεν έχει οριστεί banner, και ως header του άρθρου. Banner άρθρου (εικόνα 2, προαιρετικό): εμφανίζεται στην κορυφή του άρθρου· αν λείπει, χρησιμοποιείται η κεντρική. Όλες οι εικόνες μετατρέπονται αυτόματα σε .webp κατά το export/publish.
Εικόνες κειμένου: πρόσθεσε [img-instert-tag] εκεί που θέλεις μια εικόνα και επισύναψε τις εικόνες με την ίδια σειρά. 2+ δείκτες σε συνεχόμενες σειρές γίνονται carousel· μεμονωμένοι δείκτες παραμένουν ως μεμονωμένες εικόνες.
Χωρίς δείκτες; Οι επισυναπτόμενες εικόνες εμφανίζονται ως carousel στο τέλος του άρθρου.
Δημοσίευση: commit απευθείας στο main μέσω GitHub API — το workflow χτίζει μετά το άρθρο (~1 λεπτό). Εξαγωγή ZIP είναι το χειροκίνητο εναλλακτικό. Εισαγωγή ZIP: φορτώνει πίσω στη φόρμα ένα ZIP που έχεις ήδη εξάγει, για να συνεχίσεις την επεξεργασία. Πάτα το για να ορίσεις το token δημοσίευσης.
